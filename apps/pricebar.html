₿ Binance Price Bar ₿

Cryptocurrency prices on MacBook Touch Bar, running real time.
The app runs in background and can be accessed anywhere by tapping the ₿ icon on the Touch Bar Control Strip.
  • Platform App: MacOs
  • Last Update: never

This Mac Os application shows the current market price for a given currency pair

Features

[x] Add new coin pair.
[x] Trigger alert.
[x] 24h volume.
[x] Chart.
[x] Change chart interval.
[x] Start at login.
[x] Show Esc.
[x] RSI 14 indicator.
[x] Reorder coin pair

Preferences To update coin pairs, tap into the B icon on menu bar, select Preferences.
Tap into icon + to add new coin pair, enter the name and then tap Save to reload the touchbar.
